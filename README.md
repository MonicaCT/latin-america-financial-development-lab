## Capability focus

This repository belongs to the Data Analytics Lab because it translates financial and sector data into structured analytical outputs.

Core capabilities represented here:

- Financial analytics.
- Productive credit analysis.
- Country and sector reporting.
- R scripting.
- Reproducible figure and table generation.

## Repository contents

```text
*.R                    R scripts for loading, analysis, and figures
```
